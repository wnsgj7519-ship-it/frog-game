개구리 파리 잡기 웹앱 배포팩

이 ZIP은 바로 배포 가능한 웹앱 파일입니다.

가장 쉬운 방법
1. github.com 로그인
2. 새 public repository 생성
3. ZIP 압축 해제 후 안의 파일 전체 업로드
4. Settings > Pages > Deploy from branch > main > Save
5. 생성된 링크를 폰에서 열기
6. 크롬 메뉴 > 홈 화면에 추가

배포용 핵심 파일
- index.html
- manifest.json
- sw.js
- icons/

이 패키지는 PWA 설치를 지원합니다.
